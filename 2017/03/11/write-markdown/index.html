Markdown 语法

发表于 2017-03-11

Markdown的目标是实现易读易写
可读性要好。一份使用 Markdown 格式撰写的文件应该是简洁干净的，可以直接以纯文本发布。起码，它看起来不会像是由许多标签或是格式指令所构成，那样会降低可读性。
Markdown 的语法全由一些符号所组成，这些符号经过精挑细选，其作用一目了然。

Markdown是一种书写的格式，其理念是能让文档更容易读、写和随意改。

特殊字符

在 HTML 文件中，有两个字符需要特殊处理： < 和 & 。 < 符号用于起始标签，& 符号则用于标记 HTML 实体，如果你只是想要显示这些字符的原型，你必须要使用实体的形式，像是 &lt; 和 &amp;。
